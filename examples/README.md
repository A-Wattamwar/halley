# Halley Examples

Instrumented sample applications used to generate real traces during development and for the public demo:

- `reasoning-agent/` — Ayush's Inference-time LLM Reasoning Agent, instrumented with `@halley/sdk`
- `rag-tutor/` — Minimal RAG chatbot with retrieval spans
- `langchain-agent/` — Stock LangChain agent wired to Halley via OpenLLMetry

Populated during Phase 3 (Weeks 5–6).
