# Halley Worker

Node.js background worker. Consumes BullMQ jobs from Redis to execute long-running operations: replay/fork of past runs, evaluation suite execution, regression alerts.

Built in Phase 5 (Weeks 9–10). See [`../docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md) §3.8 and [`../docs/ROADMAP.md`](../docs/ROADMAP.md).
