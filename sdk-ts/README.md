# `@halley/sdk`

TypeScript SDK for sending agent traces to a Halley ingester.

Wraps the OpenTelemetry JS SDK with sensible defaults for LLM/agent applications and auto-instruments common LLM libraries (OpenAI, Anthropic, LangChain, Vercel AI SDK).

Built in Phase 3 (Weeks 5–6). See [`../docs/ROADMAP.md`](../docs/ROADMAP.md).
